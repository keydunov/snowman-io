# SnowManIO

Zero maintenance power parameters checks.

[![Build Status](https://travis-ci.org/snowman-io/snowman-io.svg)](https://travis-ci.org/snowman-io/snowman-io)

## Usage

Fire command `snowman server` to run SnowManIO server.
